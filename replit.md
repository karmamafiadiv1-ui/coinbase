# Front End Login Page

## Overview
A static front-end login page styled to resemble a Coinbase sign-in interface. Built with plain HTML, Tailwind CSS (via CDN), and Font Awesome icons.

## Project Structure
- `index.html` - Main login page
- `server.py` - Simple Python HTTP server for local development (port 5000)
- `front end login` - Original source file (kept for reference)

## Running
The project runs via a Python HTTP server on port 5000:
```
python server.py
```

## Deployment
Configured as a static site deployment serving files from the root directory.
